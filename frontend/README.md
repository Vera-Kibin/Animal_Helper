# AnimalHelper Frontend

This is the frontend application for the AnimalHelper shelter map project, built with React and Vite.

## Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

## Getting Started

1. **Install dependencies**
   Open your terminal in the `frontend` directory and run:

   ```bash
   npm install
   ```

2. **Run the development server**
   Start the Vite development server:

   ```bash
   npm run dev
   ```

3. **Open the app**
   Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## Backend API

Make sure the backend API is running locally (usually on `http://localhost:8000`) so that the frontend can fetch shelters and comments data successfully.

## Build for Production

To create a production build:

```bash
npm run build
```

This will generate optimized static files in the `dist` folder.
To preview the production build locally:

```bash
npm run preview
```
